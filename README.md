# decode-for-humans

Turn code into plain English — for the people who don't speak it.

`decode-for-humans` is a tool that reads source code files and explains 
what they do in plain language, producing a clean PDF that anyone can 
understand — no programming knowledge required.

---

## Status

This project is in early planning and development. Nothing is built yet.
Check back soon or watch the repo for updates.

## The Idea

Most code is written by technical people and read only by other technical 
people. But code makes decisions, handles data, and affects real workflows 
— and the people responsible for those things often can't read it.

`decode-for-humans` bridges that gap.

## Planned Features

- Plain-English PDF output for most common programming languages
- Support for notebooks (Jupyter, Quarto, RMarkdown, Marimo, Pluto)
- Fully offline with optional AI enrichment
- Simple desktop GUI — no terminal required
